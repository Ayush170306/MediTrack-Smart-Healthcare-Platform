# MediTrack: AI-Driven Smart Healthcare Platform

## Project Overview
A scalable full-stack healthcare ecosystem engineered on a modular MERN architecture. The platform integrates a TensorFlow Deep Neural Network (DNN) and a Random Forest Classifier to evaluate multidimensional user health parameters for predictive diagnostic tracking.

## Technical Stack & Frameworks
* **Frontend/Backend:** React.js, Node.js, Express.js, MongoDB (MERN Stack)
* **Machine Learning Stack:** TensorFlow, Scikit-Learn, Python
* **API Integrations:** External Health Data Streams, High-Throughput REST Endpoints

## Deep Learning & Optimization Metrics
* **Model Baseline:** Sequential Deep Neural Network (DNN) trained with Binary Cross-Entropy loss functions.
* **Optimization Framework:** Adam Optimization algorithm for dynamic learning-rate weight convergence.
* **Vetted Benchmarks:** Achieved an overall classification Accuracy of 92% and an F1-Score of 0.89 during validation tracking.

## System Architecture & Workflow
1. **Secure Client Layer:** Component-driven user interface built using React.js to capture dynamic symptomatic parameters.
2. **Decoupled API Routing:** High-throughput RESTful backend endpoints built with Express.js to process inbound health requests.
3. **Data Persistency:** Non-relational validation pipelines built using MongoDB to log user history and state tracking parameters securely.
4. **Predictive Ingestion:** Model routing pipelines via Scikit-Learn and TensorFlow to return localized health indicators instantly.
